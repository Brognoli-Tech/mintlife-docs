## Comece em três passos

Coloque seu site de documentação funcionando localmente e faça sua primeira personalização.

### Passo 1: Configure seu ambiente local

<AccordionGroup>
  <Accordion icon="copy" title="Clone seus documentos localmente">
    Durante o processo de integração, você criou um repositório GitHub com o conteúdo dos seus documentos, se ainda não tivesse um. Você pode encontrar um link para este repositório no seu [painel](https://dashboard.mintlify.com).
    
    Para clonar o repositório localmente para que você possa fazer e visualizar alterações nos seus documentos, siga o guia [Clonando um repositório](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) na documentação do GitHub.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Inicie o servidor de visualização">
    1. Instale o CLI do Mintlify: `npm i -g mint`
    2. Navegue até o diretório dos seus documentos e execute: `mint dev`
    3. Abra `http://localhost:3000` para ver seus documentos ao vivo!
    
    <Tip>Sua visualização atualiza automaticamente conforme você edita os arquivos.</Tip>
  </Accordion>
</AccordionGroup>

### Passo 2: Faça o deploy das suas alterações

<AccordionGroup>
  <Accordion icon="github" title="Instale nosso aplicativo GitHub">
    Instale o aplicativo GitHub do Mintlify no seu [painel](https://dashboard.mintlify.com/settings/organization/github-app).
    
    Nosso aplicativo GitHub faz o deploy automaticamente das suas alterações para o site de documentação, então você não precisa gerenciar os deploys sozinho.
</Accordion>
<Accordion icon="palette" title="Atualize o nome e cores do seu site">
    Para uma primeira alteração, vamos atualizar o nome e as cores do seu site de documentação.

    1. Abra `docs.json` no seu editor.
    2. Altere o campo `"name"` para o nome do seu projeto.
    3. Atualize as `"colors"` para combinar com sua marca.
    4. Salve e veja suas alterações instantaneamente em `http://localhost:3000`.

    <Tip>Tente alterar a cor primária para ver uma diferença imediata!</Tip>
  </Accordion>
</AccordionGroup>

### Passo 3: Coloque no ar

<Accordion icon="rocket" title="Publique seus documentos">
  1. Faça commit e push das suas alterações.
  2. Seus documentos serão atualizados e estarão no ar em instantes!
</Accordion>
